# Relax Map Front

Frontend частина застосунку для пошуку, перегляду та публікації локацій відпочинку в Україні.

## Про проєкт

Relax Map допомагає користувачам:

- знаходити місця відпочинку за назвою, регіоном і типом;
- переглядати детальну інформацію про локації;
- реєструватися, авторизовуватися та керувати власним профілем;
- додавати власні локації та залишати відгуки.

## Основні можливості

- головна сторінка з популярними локаціями та відгуками;
- каталог локацій із фільтрацією, пошуком і пагінацією;
- сторінка детальної інформації про локацію;
- профіль користувача та список його локацій;
- створення й редагування власних локацій;
- реєстрація, логін, логаут і перевірка сесії;
- додавання відгуків для авторизованих користувачів.

## Запуск локально

1. Встановити залежності:

```bash
npm install
```

2. Додати файл `.env.local`:

```env
NEXT_PUBLIC_API_URL=http://localhost:3000/api
NEXT_PUBLIC_APP_URL=http://localhost:3001
```

3. Запустити frontend:

```bash
npm run dev
```

4. Відкрити [http://localhost:3001](http://localhost:3001).

## Супутня інформація

- застосунок використовує `next/image`, CSS Modules і mobile-first підхід;
- для коректної авторизації бекенд має повертати cookies;
- сторінки для обробки помилок знаходяться в `app/error.tsx` та `app/not-found.tsx`.
